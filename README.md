# 2209-wns-lovelace-wild-ticket

## Installation

### Back-end (Node.js API)

Install dependencies:

```sh
cd back-end
```
```sh
npm install
```

Start server in development mode:

```sh
npm run start:watch
```

### Web app front-end (Create React App)

Install dependencies:

```sh
cd web-app
```
```sh
npm install
```

Start server in development mode:

```sh
npm run start
```
